Overview

ItemFinder adds a command to find items that you have misplaced. It will search in inventories, on belts, on the ground as entities, as items held in inserters, and - if the item corresponds to a placeable entity - for entities placed with that item. For example, if you know you crafted 150 construction robots, but there are only 148 in the roboport, this command will allow you to find the missing two, or know for certain that they have been deleted.

Usage

Simply run /findItem [item] in your chat box. This command will not count as cheating or disable achievements. The command will highlight each found instance of the item in-world with an circle overlay (akin to OrphanFinder), and on the map with a custom alert icon. Additionally, the type, location, and container will be printed to chat.
