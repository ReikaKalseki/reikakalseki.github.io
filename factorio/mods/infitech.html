Overview

InfiTech expands upon the infinite research system, so that more kinds of technologies (including modded ones) have infinite variants.

For versions of the game predating infinite research, it adds pseudo-infinite research for many of those same technologies.

Evo Reduction

There is also an optional "evolution reduction" tech, which reduces the biter evolution factor each time it is researched (becoming less effective as the evo factor approaches 100%).

Configuration Settings

This mod contains several config options to modify values and behaviors - such as for balance or server safety concerns, or simply personal taste - to your preferences. Note that several settings may have bounds imposed on their values, either logically or explicitly; these can usually be seen in the actual settings files or ingame. These are listed in order found in the source code, which is generally the order in which they were added; They may be present in a different order ingame or in the configuration files.
The settings for the mod, when installed, can be found ingame, in the mod settings menu.

Enable evolution factor research

Type: bool
Current Default Value:
true
Enable evolution factor research; each time this research is completed the evolution factor is slightly reduced.
