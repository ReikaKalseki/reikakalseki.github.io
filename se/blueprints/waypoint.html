General

This station is designed to be a space-based base or outpost, designed primarily as an intermediate destination (hence "waypoint") and cargo relay station between various routes and ship types.

In particular, it is equipped with massive storage and ore processing capability, allowing it to excel as the dropoff point for space-based miners, with the refined material (or produced components) able to be collected by other ships. It also has a large grinder array on one end, granting it the ability to rapidly salvage even large ships and then provide the resultant components for collection.

The station has tens of MW of solar capacity and an extensive battery array, and is scripted to never allow the reactors to be drawn upon unless the total demanded power outstrips the capabilities of the renewable sources.

There are also many other scripts of mine running on it, including the O2 Controller, Cargo Manager, Power Manager, Airlock Controller, and Gas Compressor Manager.

This station is another case of my redesigning a workshop blueprint; the outer hull based on the "Void Station" design, but like in every other case it has been completely gutted and redesigned. This time significant external changes were also made, including the conversion of all landing pads into ore processing wings, the extensive solar array, and a new light kit. Overall, however, I kept the same basic layout and the same aesthetic design.

This blueprint uses pure heavy armor since it is designed to be left unattended for extended periods, and given that it lacks any sort of shielding, benefits from that durability in case of pirate attack. Given that it is space-based, the mass difference is irrelevant, but if cost is an issue, you can convert it to light armor with SEToolbox or similar.

I preserved and expanded the original blueprint's very clever way of using gravity generators to create a multiplanar structure, ie one where the "down" reference point varies with location and where you can run around in three dimensions. Be aware that that can be...disorienting.

Mods

A few mods were used in the making of this station, but aside from my own mods, none of them are required for core functionality, though removing them will leave a few holes that you will want to fix.
  • Expanded Ore Processing - this provides all the specialized ore processing blocks, without which the station will not be able to process anything but magnesium, gold, and silver (unless sorters are programmed, losing efficiency and/or speed for the remaining ores)
  • Gas Compressors - this provides the bulk of the gas storage capacity; the station can be used without this, but it will have less than a fifth its usual capacity for H2 and O2.
  • Nanite Control Facility - entirely optional
  • PassageIntersections - without this, some of the entrance areas in the ore processing wings will be an empty hole
  • Twin Grinders - replace with vanilla grinders (for a speed/AoE penalty) if desired
